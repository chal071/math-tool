# Math Tool 📐

Una herramienta interactiva para aprender matemáticas.

## Funciones
- Calculadora de integrales
- Calculadora de derivadas
- Resolver ecuaciones
- Gráfico de funciones

## Tecnologías
- Python
- Streamlit
- SymPy
- NumPy
- Matplotlib

## Cómo ejecutar
``pip install streamlit sympy numpy matplotlib``

``streamlit run app.py``
